# Generative AI Projects — Muhammad Anas Naveed
Generative AI coursework projects from LUMS — Transformers, NanoGPT, LoRA  fine-tuning, BPE tokenization, and Word2Vec (SGNS), all implemented from  scratch in Python and PyTorch.

## Projects
- **NanoGPT** — Character-level language model trained on Shakespeare
- **Transformer from Scratch** — Mini LM with self-attention in PyTorch
- **LoRA Fine-tuning** — Low-Rank Adaptation applied to a causal LLM
- **BPE Tokenizer** — Byte Pair Encoding implemented from scratch
- **Word2Vec (SGNS)** — Skip-gram with Negative Sampling on LOTR text
- **Classical ML** — Decision Trees, Logistic Regression, Neural Networks
